Architecture

┌────────────┐ ┌──────────┐ ┌───────────┐ ┌──────────┐ │ Scraper │───▶│ FastAPI │───▶│ Analyse │───▶│Dashboard │ │ (httpx + │ │ REST API │ │ pandas + │ │ Streamlit│ │ BS4) │ │ :8000 │ │ sklearn │ │ :8501 │ └────────────┘ └──────────┘ └───────────┘ └──────────┘ │ │ └────────────────── Données ─────────────────────┘ (JSON)

Fonctionnalités

🕷️ Scraper

Multi-sources (Remotive), extraction automatique des compétences depuis les descriptions

🌐 API REST

FastAPI avec pagination, filtres par compétence/source, recherche plein texte

🧠 Analyse ML

TF-IDF pour les tendances, clustering K-Means avec visualisation PCA

📈 Dashboard

Streamlit interactif : graphiques, métriques, exploration des données

Stack technique

Scraping     → Python / httpx / BeautifulSoup
API          → FastAPI
Analyse      → Pandas / scikit-learn (TF-IDF, K-Means, PCA)
Dashboard    → Streamlit / Plotly
Tests        → pytest

Démarrage rapide

pip install -r requirements.txt
python -m scraper                    # Scraper les données
uvicorn api.main:app --port 8000     # Lancer l'API
streamlit run dashboard/app.py       # Lancer le dashboard
